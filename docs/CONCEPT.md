# Концепция: AnalizIstochnikov

**v6.0.1 · 2026-06-11**

> Личный research-агент для инвестора. Telegram-бот, который смотрит YouTube-видео за тебя и присылает дайджест с action items.

---

## Проблема

Ты — частный инвестор, следишь за 5–10 YouTube-блогерами по теме инвестиций (Мрочковский, Бурлаков, и т.д.). Чтобы быть в курсе, нужно:

- Каждую неделю открывать YouTube, проверять новые видео у всех
- Смотреть 20–60 минут на каждый ролик
- Запоминать, кто что сказал, какие позиции рекомендует
- Сверять с собственным портфелем
- Не забыть про налоги, ИИС, сроки отчётности

**Цена без системы:**

| Потеря | Стоимость |
|--------|-----------|
| Время на просмотр | 20–25 часов в месяц |
| Пропущенные инсайты | невозможно оценить, но это **деньги** |
| Забытые action items | ~50% рекомендаций теряются за неделю |
| Контекст между видео | теряется — не помнишь, что блогер говорил 3 видео назад |

**Попытки решения и почему они не работают:**

- *«Буду смотреть по выходным»* — накапливается 5–8 часов видео, невозможно
- *«Подпишусь на email-рассылку блогеров»* — никто из них не делает
- *«Платить ассистенту 30к₽/мес»* — для одного инвестора дорого
- *«Делать заметки вручную»* — хватит на 2 недели, потом бросишь
- *«Промптить ChatGPT по каждому видео самому»* — работает, но 30 мин на каждое

## Решение

**Telegram-бот, который ты добавляешь в контакты и дальше:**

1. Кидаешь ему YouTube-ссылку (или голосовое с названием видео)
2. Через 25 секунд получаешь структурированный дайджест: summary, action items, метаданные
3. Каждый action item можно отметить как «сделал» / «отложил» / «пропустил» — бот запоминает
4. Раз в неделю (или по настройке) — обзор недели: что было, какие паттерны, какие источники «работают»
5. Через 3 месяца — у тебя **knowledge base** по 50+ видео, доступная поиском

**Цена с системой:**

| Статья | Стоимость |
|--------|-----------|
| Время на чтение дайджестов | ~2.5 часа в месяц |
| Стоимость LLM (YandexGPT-lite) | ~15–25₽/мес |
| VPS (уже есть Beget) | 0₽ (используется существующий) |
| **Экономия времени** | **~22 часа/мес** |
| **Стоимость** | **~25₽/мес** |
| **ROI** | **время × ставка vs 25₽ — окупается в первую неделю** |

## Целевая аудитория

**Идеальный пользователь** (один):
- Частный инвестор или самозанятый (НДФЛ 13%)
- Портфель от 500к₽ до 50М₽
- 5–10 любимых YouTube-блогеров по теме
- Использует Telegram ежедневно
- Не хочет / не умеет кодить
- Не доверяет LLM «из коробки» — нужна персонализация под свой портфель
- Готов потратить 15 минут на настройку профиля

**НЕ целевая:**
- Профессиональный трейдер (нужны real-time данные, не видео)
- Мультипользовательский SaaS (security overhead, дорого на 1 vCPU)
- Крипто-трейдер (YouTube-блогеров по крипте мало, источники другие)
- Англоязычный пользователь (YandexGPT-lite оптимизирован под русский)

## Value Proposition (3 уровня)

**1. Функциональный (что делает):**
> «Telegram-бот, который за 25 секунд превращает YouTube-видео в 5 буллетов + 3 конкретных действия по твоему портфелю.»

**2. Эмоциональный (что чувствуешь):**
> «Ты в курсе, что говорят 5–10 блогеров, которых ты уважаешь, не тратя на это ни минуты. Решения принимаешь на основе инсайтов раньше, чем они становятся очевидны для рынка.»

**3. Социальный (что это даёт в долгую):**
> «Через год у тебя личная база знаний по 200+ видео, 50+ конкретных рекомендаций, ретроспектива «кто из блогеров был прав» — это инвестиционное преимущество, которое копится.»

## Границы (что НЕ делает)

Честно, чтобы не было сюрпризов:

- ❌ **Не торгует сама.** Action items — это подсказки. Решения принимаешь ты.
- ❌ **Не смотрит видео вместо тебя на 100%.** Если 90% видео — болтовня, LLM выдаёт generic. Зависит от качества блогеров в твоём пуле.
- ❌ **Не заменяет брокера.** Не знает твою реальную позицию, налоги, ИИС-лимиты.
- ❌ **Не следит за real-time новостями.** Только по новым видео.
- ❌ **Не работает с английскими блогерами** (v1). YandexGPT-lite оптимизирован под русский.
- ❌ **Не работает с криптой/форексом** (v1). Только Мосбиржа/СПБ/ОФЗ/акции РФ.
- ❌ **Не даёт «сигналов» в классическом смысле** (купить/продать сейчас). Даёт «изучи это» / «проверь то».

## Критические предположения

Без них продукт не работает. Проверить перед запуском:

1. ✅ YouTube Data API v3 ключ есть и валиден
2. ✅ YandexGPT API ключ + folder_id есть
3. ✅ Telegram-бот @ZhukovsFirstBot зарегистрирован через @BotFather
4. ✅ VPS Beget (1 vCPU, 1.9 GB RAM) — Flask помещается
5. ✅ YouTube CDN банит VPS-IP (подтверждено: `googlevideo.com = 000`) — решено через Piped
6. ⚠️ Piped-зеркала живы (community-hosted, могут падать) — есть 4 mirror + Invidious fallback
7. ⚠️ YandexGPT OAuth продолжает работать (нет deprecation в планах)

## Позиционирование vs альтернативы

| Альтернатива | Позиция AnalizIstochnikov |
|--------------|---------------------------|
| ChatGPT вручную | Дешевле (2₽ vs 30₽/дайджест), быстрее (пакетно), с историей |
| Копирайтер-ассистент | Не нужен, всё автоматизировано, нет «забыл» |
| Readwise + ручное чтение | Автоматический сбор, не нужно искать, персонализация |
| Substack / Telegram-каналы | Не зависишь от качества канала, свои правила, своя KB |
| Самописный бот | Уже готов, MIT, документирован, не нужно 200 часов |

**Главный дифференциатор:** это **не AI-ассистент «для всех»**, а **личный research-агент под твой портфель**. Персонализация через `user_profile.json` + история KB = уникальная ценность.
