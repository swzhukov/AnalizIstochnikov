## 💡 Value Proposition (3 буллета для лендинга)

**1. Экономь 22 часа в месяц**
> 20–60 минут на видео → 3 минуты на чтение дайджеста. 8 видео в неделю × 5 нед = 22 часа экономии.

**2. Не пропускай инсайты**
> Ты в курсе, что говорят все 5–10 блогеров, на которых подписан, не открывая YouTube. Решения на основе инсайтов раньше рынка.

**3. История, которая копится**
> Через 3 мес — 50+ дайджестов в личной KB. Через год — ретроспектива «кто был прав». Это инвестиционное преимущество.

## 🔧 Как это работает (4 шага)

```
┌─────────────┐   ┌─────────────┐   ┌─────────────┐   ┌─────────────┐
│  1. Кидаешь │ → │  2. Бот за  │ → │  3. Получаешь│ → │  4. Отмечаешь│
│   ссылку    │   │   25 сек    │   │   дайджест   │   │   actions   │
└─────────────┘   └─────────────┘   └─────────────┘   └─────────────┘
   Telegram         YandexGPT         HTML + 3           ✅ / ⏭ / ⏰
   webhook          Piped API         action items
                    Flask KB
```

**Шаг 1:** Отправляешь боту YouTube-ссылку (или голосовое с названием).
**Шаг 2:** Бот через Piped берёт субтитры, через YandexGPT делает summary + action items, через SQLite сохраняет в KB.
**Шаг 3:** Получаешь HTML-файл (для архива) + TL;DR текстом + inline-кнопки.
**Шаг 4:** Нажимаешь «✅ Сделаю» / «⏭ Пропустить» / «⏰ Напомни завтра» — бот запоминает.

## 🔒 Безопасность и приватность

- ✅ Auth gate: только твой `Telegram user_id` в `ALLOWED_TELEGRAM_USERS`
- ✅ Секреты в `.env`, не в коде, не в репо
- ✅ Self-hosted на VPS: данные не уходят к BigTech (кроме LLM и Telegram)
- ✅ Daily budget cap: 200₽ (защита от runaway cost)
- ✅ Rate limit: 10 дайджестов/час на user
- ✅ systemd OOM protection: Flask не убьёт n8n

**Что видит Telegram:** сообщения с URL и кнопками. **Что видит YandexGPT:** субтитры видео (только при обработке). **Что видит Piped:** только URL. **Никто не видит** твой портфель, баланс, брокер.

## ❓ FAQ (top 5)

**Q: Это безопасно? Что если бот «сольёт» мои данные?**
A: Self-hosted на твоём VPS. Внешние сервисы (YandexGPT, Telegram) не получают ничего, кроме субтитров видео и текста сообщений. Auth gate блокирует всех, кроме твоего user_id.

**Q: А если YandexGPT отключат?**
A: В v6.3 добавим LLM provider abstraction (OpenAI, Anthropic, Ollama). Сейчас — single point of failure.

**Q: Сколько видео можно обработать в день?**
A: При бюджете 200₽/день и стоимости 2₽/дайджест — до 100 видео. На практике 1–3 видео/день.

**Q: Бот сам торгует?**
A: Нет. Даёт рекомендации, решения принимаешь ты.

**Q: Можно использовать для других источников (Telegram-каналы, Substack)?**
A: Архитектура модульная, в v6.2 добавим. Пока — только YouTube.
