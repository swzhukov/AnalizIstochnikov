## 🔴 Запрещено (NO LIST)

| Нельзя | Замена |
|--------|--------|
| `fileOperations` | — |
| `readWriteFile` для записи | `HTTP POST /upload` или `/save_text` |
| `fs` в Code | Flask endpoints |
| `process.env` в Code | `$env.VAR` |
| `operation: "download"` | `resource:"file", operation:"get", download:true` |
| `localhost` / `127.0.0.1` / `172.17.0.1` | `http://host.docker.internal:8080` |
| `$credentials.*` в Set | — |
| блок `credentials: {}` в JSON | инструкция «привяжи вручную» |
| `docker exec n8n-n8n-1 bash/curl` | тестируй через HTTP-ноды (image minimal) |
| `newton fetch` для YouTube | Piped API / YouTube Data API v3 |
| `yt-dlp` + cookies | 🚫 бан аккаунта |
| `SplitInBatches` с LLM concurrency>1 | OOM kill на 1 vCPU / 1.9 GB |
| `pkill -9 -f newton-api.py` | `systemctl stop newton-api` (LESSONS §2.3) |
| Один длинный POST в `/yagpt_summarize` | chunking внутри, см. `packages/research/config.py` |
| Без `X-Telegram-User-Id` в HTTP | 403 unauthorized (v6.0.1 auth gate) |

## 🟡 Сеть и пути

| Что | Значение |
|-----|----------|
| n8n → Flask | `http://host.docker.internal:8080` |
| extra_hosts в compose | `"host.docker.internal:host-gateway"` ✅ есть |
| Файлы в n8n | `/opt/newton-tmp/` |
| Файлы на хосте | `/opt/beget/n8n/newton-tmp/` |
| Flask-скрипт | `/opt/beget/n8n/newton-api.py` |
| Лог Flask | `/opt/beget/n8n/api.log` |
| Postgres user | `user` (НЕ `n8n`!) |
| **YouTube** | `googlevideo=000` — НЕ качает. Только Data API v3 + Piped |
| **Piped** | `https://pipedapi.kavin.rocks/streams/<id>` ✅ |
| **Invidious** | `https://yewtu.be/api/v1/videos/<id>` ✅ |

**Проверка:**
```bash
curl -s http://localhost:8080/                       # с хоста
curl -s -o /dev/null -w "%{http_code}\n" \
  https://pipedapi.kavin.rocks/streams/dQw4w9WgXcQ  # Piped жив?
```

## 🟣 Piped API (субтитры без бана)

```
GET https://pipedapi.kavin.rocks/streams/<video_id>
→ { title, description, uploader, uploadDate, duration,
    subtitles: [{ url, mimeType, name, code, autoGenerated }] }
```
Фолбэки: `pipedapi.adminforge.de`, `watchapi.whatever.social`, `pipedapi.in.projectsegfau.lt`
Invidious fallback: `https://yewtu.be/api/v1/videos/<id>` → **схема другая** (`captions[]`)

**Backoff ladder (см. §4.8 в KB):**
- 429 → 1s → 2s → 4s → 8s (макс 4 retry)
- Потом rotate Piped instance
- Потом Invidious
- Потом fail с `source: 'piped_all_429'`

## 🆘 Частые ошибки → фикс (нормализованный формат)

**Формат:** `EN: <строка> / FIX: <паттерн>`

| EN-строка | FIX |
|-----------|-----|
| `operation 'download' is not supported` | `resource:"file", operation:"get", download:true` |
| `expects binary file 'data'` | `download:true` |
| `File is not writable` (readWriteFile) | `/upload` или `/save_text` |
| `Module 'fs' is disallowed` | Flask endpoints |
| `JSON Body is not valid JSON` | Объект `={{ { key: value } }}` |
| `connection refused` к Flask | iptables + extra_hosts |
| `host.docker.internal` не резолвится | extra_hosts в compose, restart |
| `OCI runtime exec failed: curl: not found` | **Не exec!** Тестируй через HTTP-ноду |
| `googlevideo.com` timeout / 000 | Piped API или YouTube Data API v3 |
| `role "n8n" does not exist` | psql `-U user`, не `-U n8n` |
| OOM kill после LLM-батча | concurrency=1, n8n Schedule Trigger с jitter |
| `youtube-transcript-api` бан | Piped API (см. §3.8 в KB) |
| `429 Too Many Requests` от Piped | Backoff 1→2→4→8s → rotate → Invidious → fail |
| `401 Unauthorized` от YandexGPT | IAM-токен через `yc iam create-token` (12ч TTL) |
| `400 x-folder-id required` | Добавить `x-folder-id` header в HTTP Request |
| Flask не стартует после изменений | §10.2 runbook в KB |
